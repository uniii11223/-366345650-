## 📋 文档格式规范

所有收录文档请遵循以下格式：

```plaintext
【问题】问题核心描述
【环境】系统/软件版本
【解决方案】详细操作步骤
【贡献者】@群友昵称
【收录日期】2026.XX.XX
```

示例

```plaintext
【问题】无障碍病毒如何解决
【环境】安卓手机
【解决方案】
1. 长按电源键+音量下键10s强制关机
2. 长按电源键+音量上键进入恢复模式
3. 进入安全模式卸载病毒应用
4. 正常重启手机
【贡献者】@小满
【收录日期】2026.03.15
```
